# 시니어 인재풀 등록 페이지 UIUX 개선 기획안
**화면 생성용 스펙 문서 · v1.0**

---

## 1. 전체 화면 구조

단일 스크롤 모달 → **4단계 Step Wizard** 로 재편

```
[STEP 1] 기본 프로필  →  [STEP 2] 커리어 설정  →  [STEP 3] 자기 소개  →  [STEP 4] 공개 설정
```

- 상단 고정: 서비스 로고 + 현재 단계명 + 닫기(X) 버튼
- Progress Bar: 4단계 표시, 완료 단계 ✓ 처리
- 하단 고정: [이전] [다음 / 저장하기] 버튼

---

## 2. STEP 1 · 기본 프로필

### 필드 목록

| 필드 | 타입 | 필수 | 비고 |
|---|---|---|---|
| 이름 | Text Input | ✅ | |
| 전문 직함 | Text Input | ✅ | placeholder: `CTO / 기술 전략가` |
| 연령대 | Dropdown | ✅ | 30대 / 40대 / 50대 / 60대 이상 |
| 최종학력 | Text Input | — | placeholder: `대학교` |

### 유효성 검사

- 포커스 아웃(Blur) 시점에 검사
- 오류: 빨간 테두리 + ❌ 아이콘 + 구체적 안내문
  - 예: `"전문 직함을 입력해 주세요. 예: CTO, 기술 전략 임원"`
- 성공: 초록 테두리 + ✅ 아이콘

---

## 3. STEP 2 · 커리어 설정

### 3-1. 희망 직종

- 대분류 탭 6개 → 탭 선택 시 하위 직종 태그 표시 (점진적 공개)
- 상단 검색창: 직종명 타이핑으로 즉시 필터
- 선택된 태그: 채운 배경 + 흰 글씨 + ✓
- 선택 항목 칩(Chip) 영역을 상단에 고정 표시

**대분류 탭 구성**

| 탭 | 포함 직종 예시 |
|---|---|
| 경영·전략 | 경영 전략, 사업 개발, 연구개발, IT·디지털, PM·PMO |
| 재무·법무 | 재무·회계, 법무·준법, 감사·리스크, 구매·SCM |
| 영업·마케팅 | 영업, 마케팅, 홍보·대외협력, 고객 경험, 고객 상담 |
| 운영·물류 | 총무·운영, 물류·유통, 생산·품질, 안전·환경 |
| HR·교육 | 인사·조직, 교육·멘토링, 강의, 심사·평가, 전문위원·고문 |
| 기타 | 서비스 기획, 건설·시설, 무역·해외사업, 사무 지원, 돌봄·케어 외 |

### 3-2. 경력 연수

- 선택 칩: 신입 / 1년 / 3년 / 5년 / 10년 / 15년 / 20년 / 30년 이상
- 직접 입력 필드 병행 제공

### 3-3. 희망 형태

- 선택 칩 (다중 선택 가능)

| 그룹 | 항목 |
|---|---|
| 정규직 계열 | 정규직, 계약직, 경력연장형 정규직, 경력연장형 계약직 |
| 프로젝트 계열 | 프로젝트 계약, 프로젝트 자문 |
| 자문·위원 | 자문/고문, 전문위원, 사외 자문, 사외이사 |
| 교육·코칭 | 멘토/코치, 강의/교육, 심사/평가위원 |
| 유연 근무 | 파트타임, 프리랜서, 업무위탁, 재택/원격, 협의 가능 |

### 3-4. 근무 가능 지역

- 선택 칩 (다중 선택 가능)
- 항목: 전국 / 서울특별시 / 부산광역시 / 대구광역시 / 인천광역시 / 광주광역시 / 대전광역시 / 울산광역시 / 세종특별자치시 / 경기도 / 강원특별자치도 / 충청북도 / 충청남도 / 전북특별자치도 / 전라남도 / 경상북도 / 경상남도 / 제주특별자치도

## 4. STEP 3 · 자기 소개 _(스킵 가능)_

### 4-1. AI 자기소개 생성 카드 (화면 상단 전체 폭)

```
┌─────────────────────────────────────────────┐
│  ✨ AI 추천                                  │
│  30초 만에 전문가다운 자기소개 완성           │
│  보유 스킬을 입력하면 AI가 맞춤 자기소개를    │
│  자동으로 작성해 드립니다.                    │
│                                             │
│  [ ✨ AI로 자기소개 작성하기 ]  ← full-width  │
└─────────────────────────────────────────────┘
```

- AI 생성 완료 후: 결과 텍스트 미리보기 + [수정하기] [다시 생성] 버튼

### 4-2. 보유 스킬

- Text Input: `DX전략, AI, 클라우드, 리더십` 형태 자유 입력
- AI 생성 시 이 필드 값을 컨텍스트로 활용

### 4-3. 자기소개 텍스트 영역

- 추천 키워드 태그 클릭 시 텍스트 영역에 자동 삽입
- 추천 태그: `책임감 있게 일합니다` / `오래 근무할 수 있습니다` / `고객 응대가 익숙합니다` / `협업이 원활합니다` / `시간 약속을 잘 지킵니다`
- 하단 글자 수 카운터: `0/300자` (실시간)
- placeholder: `'AI 자동완성' 버튼을 누르거나 직접 입력하세요...`

## 5. STEP 4 · 공개 설정

### 5-1. 연락처 정보

| 필드 | 타입 | 비고 |
|---|---|---|
| 휴대폰 번호 | Text Input | 자동 하이픈 포맷팅 `010-0000-0000` |
| 이메일 주소 | Text Input | 이메일 형식 유효성 검사 |

- 안내 문구 (필드 상단 눈에 띄게): `기업에게만 공개되며 외부에 노출되지 않습니다`

### 5-2. 프로필 공개 설정

- 토글 스위치: **기업에 공개 중** ↔ **비공개로 전환**
- 공개 상태 설명: `공개 시 기업 담당자가 프로필을 검색할 수 있습니다`
- 비공개 상태 설명: `비공개 시 기업에 노출되지 않습니다`

---

## 6. 공통 인터랙션 스펙

### 저장 액션

| 상태 | 처리 |
|---|---|
| 저장 중 | 버튼 내 스피너 + `저장 중...` 텍스트 |
| 저장 완료 | 토스트 알림: `"프로필이 저장되었습니다! 기업에 공개됩니다."` (3초 후 자동 소멸) |
| 저장 실패 | 오류 원인 명시 알림 + [다시 시도] 버튼 |

### 이탈 방지

- 모달 외부 클릭 / 닫기 버튼 → 확인 다이얼로그 노출
  - `"작성 중인 내용이 있습니다. 나가시겠습니까?"`
  - [계속 작성] [나가기]

### Auto Draft (자동 임시 저장)

- 각 단계 완료 시 자동 저장
- 재진입 시: `"이전에 작성하던 내용이 있습니다. 이어서 작성하시겠습니까?"` 안내

## 7. 화면 흐름 다이어그램

```
[모달 진입]
    │
    ▼
[STEP 1: 기본 프로필]
    │ 필수 필드 유효성 통과
    ▼
[STEP 2: 커리어 설정]
    │ 희망직종 1개 이상 선택
    ▼
[STEP 3: 자기 소개] ─── [스킵] ───┐
    │                              │
    ▼                              │
[STEP 4: 공개 설정] ◄──────────────┘
    │ 연락처 유효성 통과
    ▼
[저장하기]
    │
    ▼
[완료 토스트 → 모달 닫힘]
```
